# Art Asset List
**GoedWare Game Jam #16** | *"You Can't Save Them All"*

---

## Summary

Medieval village setting inspired by Grimm brothers fairy tales. Dark, muted palette — browns, deep greens, muted reds. Candlelight and firelight as primary light sources. Pixel art style consistent with existing character sprites.

**MVP assets: 22** | **Stretch assets: 6** | **Total: 28**

---

## Art Direction

- Color palette: dark, muted — browns, deep greens, muted reds, with demonic orange/red for boss effects
- Lighting: candlelight, firelight, boss glow contrasts against dark backgrounds
- Environment: fog, twisted trees, old wooden cottages — dark fairy tale mood
- Existing sprites: `samurai.png` (Daughter), `char1.png` (Father base) — match this style
- Cutscenes: static pixel art frames with text overlays — no full animation needed
